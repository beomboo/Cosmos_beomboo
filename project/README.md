# 사주랑 (cosmos_saju)

생년월일시로 알아보는 나의 사주팔자와 오행 밸런스 — Flutter 앱.

이 폴더(`project/`)가 Flutter 프로젝트 루트입니다. 앱 전체 규칙과 기능별 경로는 저장소 루트의 아래 문서를 먼저 참고하세요.

- [../CLAUDE.md](../CLAUDE.md) — 작업 규칙
- [../PROJECT_ROUTER.md](../PROJECT_ROUTER.md) — 기능별 경로 안내
- [../PROGRESS.md](../PROGRESS.md) — 진행 로그

## 실행/빌드

이 디렉터리(`project/`)에서 실행합니다.

```
flutter analyze        # 린트
flutter test           # 테스트
flutter build apk --debug   # Android 빌드 확인
```

iOS 시뮬레이터/macOS 빌드는 현재 이 개발 환경에서 알려진 제약이 있습니다 — 자세한 내용은 [../CLAUDE.md](../CLAUDE.md#빌드린트-체크)를 참고하세요.
