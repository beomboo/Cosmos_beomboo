# UI/UX 목업 — 사주팔자 앱

`research/`의 조사 결과(경쟁앱 분석, UI/UX 트렌드, Dr.Saju 오픈소스 실측 컬러 등)를 바탕으로 만든 4가지 디자인 방향 목업입니다. 실제 개발에 들어갈 때 **UI/UX 기준(디자인 시스템의 출발점)** 으로 삼기 위해 루트 디렉터리에 독립적으로 둡니다 — `research/`는 원자료 보관용, `mockups/`는 그 자료를 바탕으로 한 산출물이라는 역할 구분입니다.

## 보는 방법
- 온라인: https://beomboo.github.io/Cosmos_beomboo/ (갤러리에서 카드 클릭)
- 로컬: 아래 4개 `.html` 파일을 브라우저로 직접 열어도 동일하게 보입니다

## 구조
```
mockups/
├── README.md
├── 01-pastel-cute.html          # 파스텔 큐트
├── 02-dark-mystic.html          # 다크 프리미엄 미스틱
├── 03-retro-arcade.html         # 레트로 아케이드 게임화
└── 04-editorial-almanac.html    # 에디토리얼 모던 만세력
```

모든 파일은 동일한 4단계 화면(온보딩 → 정보 입력 → 계산 중 → 결과)을 담고 있어 컨셉별 비교가 쉽습니다. 각 파일은 독립적으로 완결된 HTML이라 별도 빌드 없이 브라우저에서 바로 열립니다.

## 공통 화면 구성 (04_입력정보_요구사항, 05_결과화면_사례 반영)
1. **온보딩** — 앱 소개 + 시작 CTA
2. **정보 입력** — 생년월일, 양력/음력 토글, 태어난 시간(+모름 체크), 성별, 출생지(선택)
3. **계산 중** — 오행/명식 계산 애니메이션 (컨셉별로 다른 연출: 궤도 반짝임 / 슬롯 릴 / 붓 획)
4. **결과** — 사주 명식 4기둥 + 오행 밸런스 바 차트 + 영역별(애정·재물·건강·기질) 풀이 + 공유 버튼
