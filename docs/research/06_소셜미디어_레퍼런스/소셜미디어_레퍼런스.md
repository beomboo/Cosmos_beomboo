## 포스텔러 실제 게시물 콘텐츠 패턴 (최근 12건 분석)

`opencli instagram user forceteller_official`로 실제 게시물 캡션을 확인한 결과, 명확한 콘텐츠 전략이 보입니다.

### 발행 빈도
하루 2~3건씩 꾸준히 발행 (2026-07-01~07-03 사이만 6건). 카드뉴스(carousel)·단일 사진(photo)·숏폼(video) 포맷을 섞어서 운영.

### 콘텐츠 유형 4가지
1. **엔터테인먼트/퀴즈형**: "남자 얼굴로 보는 나의 사주 취향", "사주로 보는 무인도에서 살아남기" — 사주를 밈/게임처럼 소비하게 만드는 카드뉴스. 좋아요 700~1,000+ 로 가장 반응 좋음
2. **일간(일주) 밈화**: "무자일주가 단톡에서 읽씹하는 이유ㄷㄷ" — 명리학 용어(무자일주)를 밈 문구로 캐릭터화. 짧은 영상 포맷
3. **아이돌/연예인 콜라보**: "하츠투하츠(HEARTS2HEARTS) 멤버들 사주 풀이", "인간 도화살이 있다?" — 팬덤을 활용한 트래픽 유입. 유튜브 풀버전 연계로 플랫폼 간 트래픽 유도
4. **일상 운세 CTA형**: "재물운/연애운 확인하기" + "댓글로 '행복해져라' 외쳐보세요" — 매일 반복되는 루틴형 콘텐츠, 댓글 참여를 유도하는 고정 멘트로 인게이지먼트 확보

### 앱 개발에 주는 시사점
- 결과 화면의 **카피 톤이 이미 이 정도로 캐주얼/밈스러움** — "무자일주가 읽씹하는 이유"처럼 명리학 개념을 그대로 유행어/밈으로 번역하는 것이 실제로 통함
- 앱 내 결과도 "퀴즈/유형 진단" 형태(예: "나의 사주 코어는?", "눈높이 레벨은?")로 프레이밍하면 SNS 공유 전환이 잘 될 가능성 높음
- 유튜브·인스타를 앱과 연계해 "풀버전은 앱에서" 식의 후킹 구조를 마케팅 단계에서 고려할 만함

## 참고 메모
- Twitter/X는 이 환경에서 로그인 세션이 없어 조사하지 못했습니다. 트위터 상 반응(밈, 스레드형 후기)이 필요하면 Chrome에서 x.com에 로그인 후 다시 요청해주세요.
- Dribbble/Pinterest 등 이미지 중심 디자인 사이트는 JS 렌더링 의존도가 높아 텍스트 기반 조사로는 스타일 확인이 어려웠습니다. 실제 무드보드 작업 시에는 브라우저로 직접 방문해 시각 자료를 눈으로 확인하는 것을 권장합니다.
