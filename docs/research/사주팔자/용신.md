# 용신(用神) — 콘텐츠 갭

_최종 확인: 2026-07-13_

**현황**: 미구현(현재는 "우세 오행"만 다룸 — 용신과는 다른 개념). 구현 여부는 사람 결정 대기.

## 개념
"당신에게 필요한 오행은 X"라는 개인화 해석에 쓰이는 개념. 판단법이 최소 두 갈래라 학파 편차가 있다([기초개념.md](기초개념.md) 참고): 억부법(강약 균형을 찾는 오행) / 조후법(계절 춥고 더움을 조절하는 오행).

## 경쟁 앱 노출 방식 — 두 진영

### 진영 1: "용신" 용어를 그대로 쓰는 전문/특화 앱
- **사주명**: "용신(用神): 사주에서 가장 도움이 되는 기운" — 한 문장짜리 쉬운 정의로 절충.
- **개운사주-격국용신분석**: 조후론/억부론/격국론 세 갈래 용신 분석을 각각 별도 기능으로 나란히 제공 — 학파 편차에 "셋 다 보여주고 비교"로 정면 대응.
- **사주팔자(Sia Makerlab)**: 격국·용신·조후를 정식 분석 탭으로 노출, 결정론적 규칙 엔진 + AI 해설 위임 구조(우리의 "계산-해석 분리" 제안과 같은 방향).

### 진영 2: "용신" 용어 자체를 감추고 결과만 순화
- **사주한눈에**: "용신"이라는 단어가 앱 설명에 한 번도 안 나옴 — 대신 "오행 라이프 코칭(추천 음식/활동/패션/피할 것)"으로 부족한 기운 보완 기능을 그대로 수행.
- **데이사주**: "가장 적은 오행 → 개운법" 문구로, 억부법 용신의 핵심 로직(부족한 오행 보완)을 라이프스타일 팁으로 번역.

## 우리 앱에 주는 시사점
캐주얼 톤(오행 밸런스를 쉬운 말로 풀어주는 톤)이므로 **진영 2(용어 없이 순화된 개운법 형태)** 가 자연스럽게 맞는다 — "가장 부족한 오행을 색상/음식/활동으로 보완하는 팁" 형태면 억부법 하나만 골라 쓰는 데 대한 학파 논쟁 부담도 준다. 향후 전문가용 확장 화면을 만든다면 그때는 진영 1(용어 노출+이론별 비교)이 참고 대상.

십신은 "일간 기준 8글자 전체"를 분류하는 근본적인 축인 반면, 용신은 오행 밸런스 분석 결과에서 파생되는 "그래서 뭘 보완하면 좋은가"라는 한 단계 더 나아간 결론 — 이미 있는 오행 밸런스 기능에 자연스럽게 이어붙일 수 있어 십신보다 구현 문턱이 낮다.
