# 사주팔자 기초 개념

_최종 확인: 2026-07-16_

## 용어
- **사주(四柱)**: 연주(年柱)·월주(月柱)·일주(日柱)·시주(時柱) 네 기둥
- **팔자(八字)**: 네 기둥 × 천간(天干)+지지(地支) 2글자 = 총 8글자
- **만세력(萬歲曆)**: 생년월일시 → 사주팔자를 산출하는 절기·60갑자 순환 기록 도구. 앱의 핵심 계산 엔진에 해당

## 오행(五行)
목(木)·화(火)·토(土)·금(金)·수(水). 8글자가 각각 오행 하나씩에 속하며, 그 분포로 성격 경향·에너지 균형을 해석한다.

## 계산에 필요한 핵심 요소
1. 생년월일시(분 단위까지 있으면 더 정확)
2. 양력/음력 구분
3. 진태양시 보정(출생지 경도+균시차) — 자세한 크기는 [../사주팔자/계산_정확도_이슈.md](계산_정확도_이슈.md)
4. 자시(子時) 처리 방식(23:00~01:00 출생자) — 위 문서 참고
5. 성별 — 대운 순행/역행 방향 결정에 필요, [../사주팔자/대운_세운.md](대운_세운.md) 참고
6. 출생 시간 미상 시 시주 계산 불가 → 삼주(三柱)로 근사 해석하는 앱이 많음

## 지장간(支藏干)·용신(用神) 판단법 — 학파 편차
- **지장간 기준 자체가 학파마다 다르다** — "연해자평(淵海子平)" 기준이 가장 보편적이나 고전마다 조금씩 다름.
- **오행 개수 집계도 지장간 포함 여부로 갈린다** — 표면 8글자만 세는 쪽이 초보자에게 권장되는 방식.
- **용신 판단법은 실제로는 최소 5갈래(억부·조후·격국·통관·병약, 전왕까지 세면 6갈래)** — 억부법(사주 강약 균형)/조후법(계절 춥고 더움 조절)이 실무에서 가장 흔히 쓰이지만 그 둘이 전부는 아니며, 특히 격국법은 억부·조후와 아예 다른 체계(월령 중심)라 결과가 다르게 나오는 경우가 흔하다(2026-07-16 재검증). 자세한 내용은 [용신.md](용신.md).
- 지적할 만한 점: "AI로 만든 사주 앱들이 다 똑같이 생긴 이유"는 AI가 특정 유파를 선택하지 않고 평균을 뱉기 때문이라는 분석이 있음(명리학은 학파가 갈려온 학문이라 어느 기준을 택하는지가 앱의 정체성이 됨).

### 우리 코드 현재 상태
`ganzhi.dart`의 `GanzhiPillar.ohaeng`는 `[stemOhaeng, branchOhaeng]`(표면 2글자만)만 반환 — 지장간 미포함, "초보자 추천" 방식과 일치(버그 아님, 의도된 포지셔닝). 용신 개념 자체는 미구현(현재는 "우세 오행"만 다룸 — 서로 다른 개념이니 향후 확장 시 혼동 주의).

## 데이터 정확도 기준
- KASI(한국천문연구원) 정본 절기 데이터를 분 단위로 쓰는 라이브러리가 정확도 높음(예: manseryeok)
- 계산 가능 범위는 라이브러리마다 다름(보통 양력 1800~2300년, 음력 1800~2100년)

절기 정밀 계산과 무관한 5가지(공망·세운·십신·십이운성·신살)는 기존 코드베이스 그대로 구현 가능. 대운만 유일하게 정밀 절기 계산(계산_정확도_이슈.md)에 종속. 십신·용신·십이운성·신살은 모두 "심층 분석"처럼 별도 확장 화면에 배치하는 기존 정보구조 패턴이 그대로 적용 가능해 보인다.

**구현 여부·우선순위는 전부 사람 결정 대기 — 이 문서는 콘텐츠 아이디어 기록용이다.**
