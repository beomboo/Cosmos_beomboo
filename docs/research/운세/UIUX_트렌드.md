# UI/UX 트렌드

_최종 확인: 2026-07-13_

## 핵심 방향
"무거운 점술" → "가벼운 자기이해 콘텐츠". 전통 사주 상담소의 무거운 분위기를 벗고 모바일 최적화된 직관적·재미있는 경험으로 재해석하는 것이 MZ세대 공략의 핵심.

## 1. 캐릭터화
MZ세대는 캐릭터화된 운세를 선호 — MBTI처럼 "나는 어떤 동물/기운 타입일까?" 라벨링 게임 형식이 효과적. 최고심 브랜드의 스크래치 복권형 부적 카드("돈벼락"·"합격" 문구+귀여운 일러스트), 카카오·토스도 자체 캐릭터로 디지털 부적 제작.

## 2. 컬러 팔레트
밝은 원색보다 **소프트 톤(파스텔) 선호도 높음**. Z세대 팔레트: 피부/베이스=피치·소프트 핑크, 포인트=라벤더·민트 그린, 서브=세이지 그린·페리윙클, 강조(CTA)=코랄·터쿼이즈.

## 3. 언어/카피 톤
명리학 전문 용어(십신, 편관 등) 그대로 노출 금지 — 일상어로 재해석. "내 2026년 키워드는 OO" 같은 한 줄 요약형 카피가 SNS 공유에 효과적.

## 4. SNS 바이럴 설계
결과 화면을 인스타그램 스토리 규격(9:16)에 최적화. 공유 버튼은 하단에 묻히지 않게 상단/CTA 위치에 배치.

## 5. 인터랙션 요소
로딩(계산 중) 구간에 캐릭터 애니메이션 배치(대기 시간의 콘텐츠화). 대화형 인터페이스(AI 챗봇 형태)로 결과 전달도 트렌드(Dr.Saju의 8종 캐릭터 챗). 스크래치/카드 뒤집기 게임화 인터랙션으로 결과 공개.

## 6. 온보딩 vs 스토어 스크린샷
스토어 스크린샷은 "설치 유도", 온보딩 화면은 "서비스 이해·기능 활용" — 서로 다른 메시지로 설계.

## 디자인 방향성 두 갈래 — 실제 소스 코드로 확인된 대비

| | **파스텔 큐트형**(포스텔러·헬로우봇) | **다크 프리미엄 미스틱형**(Dr.Saju 실제 코드 기준) |
|---|---|---|
| 배경 | 흰색/밝은 배경 | `#0a0a0f` 거의 검정 |
| 포인트 컬러 | 파스텔(피치·라벤더·민트) | 보라 `#8b5cf6` + 골드 `#C5A44E` |
| 톤 | 귀엽고 가벼움, 밈/게임화 | 신비롭고 고급스러움 |
| 캐릭터 | 귀여운 마스코트 중심 | 캐릭터별 챗 라우트는 있지만 비주얼은 절제된 다크 UI |
| 타겟 인상 | 10대~20대 초반 | 20대 후반~30대, "믿을 만한 프리미엄" 인상 |

**우리 앱은 파스텔 큐트를 채택**(`docs/mockups/01-pastel-cute.html`).

## Dr.Saju 실제 컬러/컴포넌트 분석 (GitHub API로 소스 직접 확인)

| Primary Dark | `#7c3aed` | hover 등 |
| 포인트(골드) | `#C5A44E`("legal-gold") | 프리미엄/유료 배지 추정 |
| Background | `#0a0a0f` | 거의 검정 |

폰트 DM Sans, 모노스페이스 Geist Mono. shadcn/ui + daisyUI, radius `0.75rem` 기본. 마이크로 인터랙션: `glowPulse`(보라 글로우 펄스)·`shimmer`·`marquee`·`fadeIn/slideUp/scaleIn` 스태거 애니메이션 다수 — 정적 리포트가 아니라 등장 연출에 공들인 구조.

### 실제 사용자 플로우 (`src/components/saju/` 폴더명 기준)
```
landing → input → preview → result → report
                                 ↓
                    share / referral / upsell → payment → coin-shop
chat/[characterId] → (캐릭터별 AI 채팅 결과 해석)
```
- **result → report**: 요약형 결과 → 상세 리포트(유료)로 확장되는 지점이 명확히 분리 — 무료/유료 경계 설계에 참고([결과화면_설계.md](결과화면_설계.md)).
- **share / referral**: 결과 공유·추천인 초대가 결과 화면 바로 다음 단계 — 바이럴 루프가 플로우에 내장됨.
- **upsell → payment → coin-shop**: 코인(재화) 구매 방식 과금, 리포트 1건당 코인 차감 구조로 추정.

## 참고 경쟁 앱 UI 포인트
- **포스텔러**: 캐릭터 기반, 무료 콘텐츠 진입장벽 최소화. 리뷰 "캐릭터들도 진짜 귀엽고 사랑스럽다"
- **점신**: '운세보고서' 형식(오늘의 운세·총운·시간대별·바이오리듬·추천소식 종합), 상담사 랭킹으로 신뢰 부여
- **헬로우봇**: 흰 배경+굵은 글씨+둥근 카드, 캐릭터 '라마마'가 챗봇 형태로 대화하듯 결과 전달
- **Dr.Saju**: 8가지 캐릭터와의 AI 챗 형태(`chat/[characterId]`)로 결과 전달
