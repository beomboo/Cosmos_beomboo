## 종합 시사점 (우리 앱 전략에 적용)
1. **언어 난이도**: 명리학 전문 용어를 그대로 노출하지 말고 일상어로 재해석 — 포스텔러 성공의 핵심

3. **콘텐츠 다각화 여지**: 사주 하나로 시작하되 궁합·오늘의 운세 등으로 확장 가능한 구조로 설계
4. **SNS 연계**: 결과 공유가 곧 마케팅 — 인스타그램 스토리 공유를 처음부터 UX에 내장
5. **과금 과다는 이탈 요인** — 초반 유료 전환 압박은 최소화

## 유료 소비 성별·연령 실측 데이터 (2026-07-06 웹 검색 보강) — 05의 헬로우봇 인터뷰 교차 검증

05_결과화면_사례에서 헬로우봇 인터뷰를 통해 "사주=27세 이상 직장인(고가 지불 의향), 타로/심리테스트=Z세대(무료 위주)"라는 소비 패턴을 확인했는데, 이를 다른 출처(갤럽 설문조사)로 교차 검증했다.

- **갤럽 리서치("점·사주" 조사)**: 점·사주를 신뢰한다는 응답 72%, 신뢰하지 않는 응답자 중에서도 20%는 유료로 본 경험이 있다고 답함. **유료 경험자의 성별 분포는 여성 58% vs 남성 22%로 여성이 압도적으로 많고, 20대 남성은 유료 경험이 8%에 불과** — 헬로우봇 인터뷰의 "사주는 27세 이상이 주 소비층"이라는 진술과 함께 보면, 유료 사주 소비는 (a) 남성보다 여성이 많고 (b) 어릴수록 적다는 두 축이 모두 확인됨.
- **SBS 프리미엄 기사**: 16~24세 여성의 유료 운세 소비 비중이 2019년 33% → 2022년 27%로 매년 하락 추세 — Z세대일수록 유료 결제에서 멀어지는 흐름이 시계열로도 확인됨(같은 연령대 남성은 애초에 비중이 훨씬 낮음).
- **종합 시사점**: "무료로 낮은 진입장벽 + 유료 심화 리포트"라는 기존 전략 방향은 유지하되, 유료 전환 타겟팅을 굳이 좁힌다면 20대 초반보다는 **20대 후반 이상 + 여성** 쪽이 실제 결제 전환 가능성이 통계적으로 더 높다는 근거가 됨. 다만 이건 유료화 자체를 결정하는 게 아니라(결제 로직은 여전히 사람 결정 대기), 결정 시점에 참고할 타겟 세그먼트 힌트.

## 2026년 최신 시장 지표 (2026-07-06 웹 검색 보강, 아주경제 2026-02-13 / 한국경제 2026-02-10 기사 기준)

기존 "핵심 요약"의 시장 수치(포스텔러 900만 가입자 등)는 조사 시점이 다른 여러 출처가 섞여 있어, 가장 최근(2026-02) 기사 기준 수치를 별도로 남겨둔다 — 기존 수치를 지우지 않고 병기하는 이유는 출처·시점이 다른 숫자를 하나로 합치면 오히려 부정확해지기 때문.

- **점신**: 누적 다운로드 1,700만 건(한국경제), 2025-12 기준 MAU 79만 4,849명(전월 대비 12.1%↑, 아주경제) — 이 중 2030세대 비중 65.6%. 운영사 테크랩스는 2023년 매출 830억 원, 영업이익 100억 원(한국경제).
- **포스텔러**: 누적 가입자 750만 명(한국경제, 기존 자료의 "900만"보다 최근 수치), 2025-12 기준 MAU 62만 9,050명(전월 대비 8%↑, 아주경제) — 2030세대 비중 57%.
- **가격대 스펙트럼(아주경제)**: 기본 하루 운세는 무료, 유료는 990원 소액 상품부터 10만원대 종합 운세 패키지까지 폭넓게 분포 — 05에서 확인한 "500포스(5,000원)/22,000원" 사례와 함께 보면 실제 시장 가격대는 990원~10만원+ 로 매우 넓은 스펙트럼.
- **비대면 전문가 상담(신규 트렌드, 한국경제)**: 점신·포스텔러 외 "천명" 등 플랫폼이 명리 전문가·신점 상담가의 채팅/전화 상담을 중개 — 5~10분당 5,000~20,000원, 인기 상담가는 건당 3만원 이상. 신규 상담의 절반 이상이 비대면(일부 서비스는 70%+)으로 진행될 만큼 이미 주류화됨. 플랫폼 수수료는 상담 수익의 20~40%.

### 비대면 전문가 상담 추가 조사(2026-07-07, README "다음으로 조사가 더 필요할 만한 지점" 7번 항목) — 플랫폼별 구체 수치·신규 하이브리드 트렌드
위 항목의 "5~10분당 5,000~20,000원"은 시장 전반의 평균치였는데, 천명 개별 사례와 각 플랫폼의 실제 수익 규모, 그리고 최근 등장한 AI-사람 하이브리드 상담 흐름까지 추가로 확인했다.
- **천명 개별 가격**: 전화 통화 상담 기준 30분당 2만원~10만원(RTZR 고객 사례 인터뷰) — 위 시장 평균("5~10분당 5,000~20,000원")보다 훨씬 폭넓은 상단 가격대까지 형성돼 있음을 확인. 누적 상담 후기 25만 건 이상(2026-02 기준).
- **2023년 매출 규모 비교(머니투데이)**: 점신(테크랩스) 약 60억 원 추정, 포스텔러(운칠기삼) 약 99억 원, 천명(천명앤컴퍼니) 약 48억 원 — 상담 중개형(천명)이 광고 기반 무료형(점신)·복채 결제형(포스텔러)보다 매출 규모 자체는 작지만, 상담 건당 단가가 훨씬 높아(30분 2만~10만원 vs 일반 콘텐츠 990원~) 객단가 구조가 다른 것으로 보임. 점신의 실제 수익모델은 광고+부적판매(콘텐츠 자체는 무료)로 재확인됨.
- **"사주GPT" — AI가 사람 상담사를 대체하는 게 아니라 보조하는 하이브리드 사례(한국AI부동산신문, 2026-02-11)**: 생년월일시만 입력하면 사주를 분석해주는 AI 자동화 서비스(MAU 25만, 전년 대비 20%+ 성장)가 등장했는데, 사용자들은 이를 "상담 전 사전 참고용"으로 쓰고, 흥미롭게도 **일부 상담사 본인도 기본 운세 설명은 사주GPT를 활용하고 더 깊은 해석·고민 상담만 직접 진행**하는 방식으로 상담 효율을 높이는 사례가 확인됨 — AI가 사람 상담을 완전히 대체하기보다 "기초 설명은 AI, 심화·정서적 교감은 사람"으로 역할이 나뉘는 실제 운영 패턴.
- **우리 앱에 주는 시사점**: 결제 로직 결정 시 스펙트럼이 "AI 해석만" ↔ "실제 전문가 상담 연결"의 이분법이 아니라, 그 사이에 "AI가 기초 설명, 사람이 심화 상담"이라는 제3의 하이브리드 지점이 실제 시장에서 검증되고 있다는 점을 참고할 만함(단, 상담사 매칭·정산 인프라가 필요해 우리 앱 규모에는 훨씬 무거운 결정 — 여전히 사람 결정 대기).

## Flutter 기반 소규모 경쟁 사례 발견 — "사주냥(SAJUNYANG)" (2026-07-07, Wishket 포트폴리오 확인)

우리 앱과 정확히 같은 기술 스택(Flutter)으로 만들어진 실제 경쟁 사례를 찾았다 — 대형 회사가 아니라 **개발자 1인이 위시켓(프리랜서 매칭 플랫폼)을 통해 만든 MVP**라는 점에서, 우리 프로젝트 규모와 비교하기에 특히 참고 가치가 높다.

- **기술 스택**: Flutter(iOS/Android) + Firebase Functions(Node.js/TypeScript) 백엔드 + ChatGPT API(GPT-4o-mini) 직접 연동 — "AI 해석 연동"을 프롬프트 위임형(사용자가 직접 복사해서 외부 AI에 붙여넣는 방식, 05에서 이미 확인)이 아니라 **앱이 직접 API를 호출하는 방식**으로 구현한 실제 소규모 사례. 개발자 1인이 기획·디자인·개발·계산 로직까지 전부 수행했다는 점에서, 직접 API 연동이 우리 프로젝트 규모에서도 기술적으로는 충분히 감당 가능한 범위임을 보여주는 참고 사례(단, 실제 API 비용·트래픽 규모는 공개되지 않아 비용 데이터로 쓸 수는 없음).
- **계산 로직 범위**: "천간, 지지, 오행, 십신, 십이운성"을 직접 구현 — 십신 다음 단계로 흔히 짝지어지는 개념이 **십이운성(十二運星)**임을 새로 확인(우리가 지금까지 조사한 콘텐츠 갭 목록엔 없던 개념).
- **경쟁 앱 마케팅 문구에서 추가로 발견한 미조사 개념들**: "2026 정통사주" 앱은 "신살, 궁합, 택일까지 실시간으로 묻고 답을 듣는" GPT 챗봇을 내세우고, 별도로 확인한 사주냥 개발 상세에도 "50종 신살(神殺) 분석"이 포함돼 있음 — **신살(神殺, 길흉을 나타내는 특정 간지 조합, 50종 이상)**과 **택일(擇日, 좋은 날 고르기)**도 우리 앱엔 없는 개념. 이번 사이클에서 전부 깊이 조사하지는 않았지만(십신·용신·대운/세운·공망 4가지로 이미 콘텐츠 갭 종합을 마친 상태라 범위 확장은 신중하게), 존재 자체는 기록해둔다 — 향후 콘텐츠 확장을 고려할 때 후보군에 추가할 만함.
- **문제의식 재확인**: 위시켓 프로젝트 설명이 명시한 사주냥의 기획 배경 — "기존 사주/운세 앱은 단순 점수나 한두 줄 결과만 제공하여 신뢰도와 몰입감이 부족", "사주 데이터는 복잡한 한자와 용어로 구성되어 일반 사용자가 직관적으로 이해하기 어려움" — 이 우리 앱이 이미 추구하는 "캐주얼 톤 + 쉬운 설명" 방향과 정확히 같은 문제의식에서 출발했음을 재확인(이 접근이 우리만의 특이한 선택이 아니라 업계에서 공통으로 인식하는 문제라는 뜻).
