# 사주팔자 기초 개념 정리

## 용어
- **사주(四柱)**: 네 개의 기둥 — 연주(年柱)·월주(月柱)·일주(日柱)·시주(時柱)
- **팔자(八字)**: 위 네 기둥이 각각 천간(天干) + 지지(地支) 2글자씩 = 총 8글자
- **만세력(萬歲曆)**: 생년월일시를 기준으로 사주팔자를 산출하는 절기·60갑자 순환 기록 도구. 앱의 핵심 계산 엔진에 해당

## 오행(五行)
목(木)·화(火)·토(土)·금(金)·수(水) 다섯 가지 기운. 사주의 8글자가 각각 오행 하나씩에 속하며, 오행 분포로 타고난 성격 경향·에너지 균형을 해석.

## 십신(十神)
일간(태어난 날의 천간)을 기준으로 나머지 글자와의 관계를 10가지로 분류:
비견, 겁재, 식신, 상관, 편재, 정재, 편관, 정관, 편인, 정인 — 대인관계 스타일, 재물 성향, 직업 적성 해석에 사용.

## 계산에 필요한 핵심 요소
1. **생년월일시** (연·월·일·시, 분 단위까지 있으면 더 정확)
2. **양력/음력 구분**
3. **진태양시 보정**: 출생지 경도 + 균시차 반영 시 정확도 상승 (표준시와 실제 태양 남중 시각의 차이 보정)
4. **자시(子時) 처리 방식**: 23:00~01:00 사이 출생자는 "조자시/야자시" 처리 방식에 따라 일주가 달라질 수 있음 — 앱에서 이 부분을 어떻게 안내하느냐가 UX 디테일 포인트
5. **성별**: 대운(10년 단위 운세 흐름) 계산 시 순행/역행 방향이 성별에 따라 달라짐
6. **출생 시간을 모르는 경우**: 시주(時柱) 계산 불가 → 사주가 아닌 "삼주(三柱)"로 근사 해석하거나, 앱마다 처리 방식이 다름 (자세한 내용은 04_입력정보_요구사항 참고)

## 데이터 정확도 기준
- KASI(한국천문연구원) 정본 절기 데이터를 분 단위로 사용하는 라이브러리가 정확도 높음 (예: manseryeok)
- 계산 가능 범위는 라이브러리마다 다름 (보통 양력 1800~2300년, 음력 1800~2100년)
