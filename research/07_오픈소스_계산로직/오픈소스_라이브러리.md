# 오픈소스 사주 계산 로직 / 라이브러리

## 추천 우선순위

### 1. manseryeok (TypeScript) — 계산 엔진 후보 1순위
- https://github.com/yhj1024/manseryeok
- KASI(한국천문연구원) 정본 절기 데이터 기반, 1800~2300년(양력) 지원
- **입력**: 연/월/일/시(24h)/분, 양력·음력 선택, 진태양시 보정(경도/균시차), 자시 처리 방식, 성별(대운용)
- **출력**: 년/월/일/시주(천간지지), 오행/음양, 십신, 공망, 대운(성별 입력 시)
- 런타임 의존성 없음, ~40KB 경량 — 웹/앱 프론트엔드에 직접 임베드 가능

### 2. sajupy (Python)
- https://github.com/0ssw1/sajupy
- `pip install sajupy` → `calculate_saju(1990, 10, 10, 14, 30)` 형태로 최소 5개 인자만으로 계산
- 도시명으로 경도 자동 조회 기능 있음 (`city=` 파라미터) — 진태양시 보정 UX를 단순화하기 좋음
- 백엔드를 Python으로 가져갈 경우 적합

### 3. Dr.Saju (imgompanda/drsaju-opensource) — 풀스택 레퍼런스 아키텍처
- https://github.com/imgompanda/drsaju-opensource ⭐30
- "계산은 코드, 해석은 AI"로 역할 분리 — 우리 앱 아키텍처의 직접적인 참고 모델
- 스택: Next.js 16(App Router) + TypeScript + Supabase(Auth/DB) + Vercel AI SDK + Gemini + shadcn/ui + Tailwind CSS 4 + @react-pdf/renderer + next-intl(다국어)
- 계산은 manseryeok 라이브러리 사용, AI가 성격/재물/건강/연애 등 해석 생성
- 화면 구조: `chat`(AI 캐릭터 8종과 대화형 결과), `reading`(결과), `my-readings`(이력), `saju-report`(PDF), `coin-shop`(결제)
- 궁합 분석, 카카오톡 공유 기능 포함

### 4. bdev-saju (Claude Code 스킬 형태)
- https://github.com/be-realdeveloper/bdev-saju
- "생년월일시만 넣으면 유료 앱 수준의 18장 풀이 + 상담 모드" — 바이브코딩(AI 코딩 어시스턴트)용 Claude Code 스킬로 배포된 점이 특이
- 프롬프트/해석 텍스트 구조를 참고하기 좋음 (사주 풀이를 몇 개 챕터로 나누는지 등)
